Usage

Command-line

Either install it globally:

$ npm i -g markmap-lib
$ markmap my-markdown.md

or use with npx:

$ npx markmap-lib my-markdown.md

Programmatic usage

Make sure there is an SVG element in your DOM, say:

<svg id="mindmap" style="width: 800px; height: 800px"></svg>

then we can mount the markmap to it:

import { transform } from 'markmap-lib/dist/transform.common';
import { markmap } from 'markmap-lib/dist/view.common';

const md = `
# hello

- This is my Markdown.
`;

const data = transform(md);
markmap('#mindmap', data);
